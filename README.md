# Filmmaking Skills

Claude Code skills for film & TV production work, shared by Mo.

A **skill** is a folder of instructions that teaches Claude Code how to do a
specific job well. Drop these into your own setup and Claude will pick them up
automatically when the topic comes up.

## How to install (once skills are published)

1. Install [Claude Code](https://claude.com/claude-code).
2. Copy the skill folders you want into `~/.claude/skills/`:

   ```
   git clone https://github.com/mohilaly/filmmaking-skills.git
   cp -R filmmaking-skills/<skill-name> ~/.claude/skills/
   ```

3. Start Claude Code and just trigger the skill.
